## Image Guessing Game
A simple and interactive web-based game built to practice Frontend Development fundamentals. The goal is to guess the topic of the displayed image by typing the correct word into the input field.

## Features
Interactive Gameplay: Instant validation of user input against the current topic.

Scoring System: Points are automatically awarded for every correct match.

Win Streak Logic: Tracks consecutive correct answers in real-time.

Visual Feedback: Features a "fire" indicator (streak icon) that ignites once the user achieves a series of correct answers, implemented via dynamic CSS class toggling.

## Tech Stack
HTML5: Semantic structure for the game board and input elements.

CSS3: Interface styling, layout, and "streak fire" animations.

JavaScript (Vanilla JS):

DOM Manipulation: Real-time updates of scores and UI elements.
Event Handling: Listening for user input and triggering validation.
State Management: Tracking current scores and win streak counts.
